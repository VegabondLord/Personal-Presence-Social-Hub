# SocialHUB Website
Modern social media hub interface.

## Description
This project is a modern, responsive social hub website with various features including navigation, profile management, and interactive components.

## Installation
Just open `index.html` in your browser or serve it using a local server.

## Technologies
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- PHP (Backend components)
